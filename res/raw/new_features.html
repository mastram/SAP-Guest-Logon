    • New in Version 6.5

    In this version, many new features have been added and old bugs have been fixed.
    New Features:
  • Encryption: You can not encrypt your logon data to make it more secure.
    You can either use medium level encryption or high level encryption.
  • Auto Logon feature: You can now activate the Autologon feature.
    If this function is activated, the application logs in automatically when you enter the SAP Guest Network range.
    You do not have to launch the application manually.

    • Crash Issues Fixed
  • The application used to crash when out the device was running low on memory or sometimes when the device was on sleep.
    This issue has been fixed.

  • We have improved the security aspects of this app. The app validates all the required server certificates before posting your credentials.
