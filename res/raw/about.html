About the App

SAP Guest Logon app provides one touch logon to the SAP Guest network in SAP offices.

Disclaimer

This app is not associated to SAP in any manner.

Privacy

All the data related to this app is stored locally (in internal memory) on this mobile device. All the data related to this app will be completly wiped out when you un-install this app.

Security

This app is a secure app and it verifies the security certificates issued by SAP before posting your credentials.
The logon data is encrypted and saved in the internal memory of the mobile which is not accessible to other applications.

Analytics:

The developer reserves the rights to track the application usage statistics anonymously using the Google Analytics. The developer does this in order to constantly improve the app.
Please note:No personal information about the user is tracked. Only the application usage statistics and crash reports are tracked.
